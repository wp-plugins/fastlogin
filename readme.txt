== Description ==

Plugin for fast enter log in page with keys combo Ctrl+Shift+l

== Installation ==

1. Upload `fastlogin.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Try to enter login page from any place of your site with Ctrl+Shift+l combo
